# Getting Started Guide

Connect Camp Admin Workspace is the internal operations platform for Montgomery College Student Life.

## First Steps

1. Sign in with your admin account.
2. Review the Dashboard for current operational metrics.
3. Check Pending Approvals and Tasks at the start of each day.
4. Report issues through Support Center when needed.

## Core Workflows

- Club oversight
- Event approvals
- Member management
- Messaging and incident response
